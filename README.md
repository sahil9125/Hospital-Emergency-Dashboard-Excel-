# Hospital-Emergency-Dashboard-Excel-
This project is an interactive Hospital Emergency Dashboard built in Microsoft Excel using Power Query, Power Pivot, DAX, Pivot Tables, and Charts. The dashboard provides meaningful insights into emergency room operations and helps analyze patient flow, waiting time, admissions, referrals, and demographic trends.

Objectives
Analyze hospital emergency room data.
Monitor patient admissions and wait times.
Identify patient demographics.
Track department referrals.
Create an interactive dashboard for better decision-makin
## Project Summary

The Hospital Emergency Room Dashboard is an interactive data analytics project developed in Microsoft Excel to analyze and monitor emergency room operations. The project transforms raw hospital emergency room data into meaningful insights using data cleaning, data transformation, data modeling, calculated measures, Pivot Tables, and interactive visualizations.

The dashboard provides a comprehensive overview of patient flow and emergency room performance by analyzing key metrics such as total patients, admissions, waiting time, patient satisfaction, referral departments, patient demographics, and attendance trends. It enables users to explore patterns across different time periods, patient categories, age groups, departments, and other operational factors.

Power Query was used for data cleaning and transformation, while Power Pivot and DAX were used for data modeling and creating calculated measures. Pivot Tables, Charts, and interactive dashboard elements were used to present the analysis in a clear and user-friendly format.

This project demonstrates practical skills in Excel-based data analytics, healthcare data analysis, KPI tracking, dashboard development, data visualization, and data-driven decision-making.
