# Hospital-Emergency-Dashboard-Excel-
This project is an interactive Hospital Emergency Dashboard built in Microsoft Excel using Power Query, Power Pivot, DAX, Pivot Tables, and Charts. The dashboard provides meaningful insights into emergency room operations and helps analyze patient flow, waiting time, admissions, referrals, and demographic trends.

Objectives
Analyze hospital emergency room data.
Monitor patient admissions and wait times.
Identify patient demographics.

## Project Summary
Project Name---------Hospital Emergency Dashboard (Excel)

Project Overview

The Hospital Emergency Dashboard is an interactive Excel dashboard developed to analyze and visualize emergency room patient data. The project helps monitor hospital performance by providing key insights into patient admissions, waiting times, department referrals, patient demographics, and service efficiency.

The dataset was imported from a CSV file and transformed into meaningful business insights using Excel's advanced data analysis features.

-----------------------Tools & Technologies Used
Microsoft Excel

Used as the primary tool to build the complete dashboard.

===============================Power Query

Power Query was used for:

Importing the CSV dataset.
Cleaning the data.
Removing duplicates.
Handling missing values.
Changing data types.
Creating additional columns when required.
Transforming raw data into an analysis-ready format.
###########################################Power Pivot

Power Pivot was used to:

Create the data model.
Build relationships between tables.
Improve data organization.
Enable advanced calculations using DAX.
----------------------------------------- DAX (Data Analysis Expressions)

DAX measures were created to calculate important KPIs such as:

Total Patients
Average Wait Time
Patient Satisfaction Score
Total Admitted Patients
Total Non-Admitted Patients
Percentage of Patients Seen On-Time
Department Referral Counts

These measures made the dashboard dynamic and interactive.

---------------------------------------------Pivot Tables

Pivot Tables were used to:

Summarize patient data.
Analyze admission status.
Group patients by age.
Generate gender-wise reports.
Analyze department referrals.
Build summary reports for charts.
------------------------------------------Charts & Visualization

Different Excel charts were used to present insights clearly, including:

Column Charts
Bar Charts
Pie Charts
Doughnut Charts

These charts make the dashboard easy to understand.

--------------------------------------------- Slicers

Slicers were added to allow users to filter data by:

Year
Month

This makes the dashboard interactive and user-friendly.
